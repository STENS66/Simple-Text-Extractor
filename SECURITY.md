# Security Policy

I take the security of **Simple-Text-Extractor** very seriously. As an independent developer, I built this application with a strong focus on privacy (offline operation, local processing). Therefore, I am committed to promptly resolving any security issues that could compromise the application's integrity or user experience.

## Supported Versions

Only the versions listed below currently receive security updates.

| Version | Supported | Notes |
| :--- | :--- | :--- |
| 1.1.x | ✅ | Current version (Microsoft Store) |
| < 1.1 | ❌ | End-of-Life / Obsolete |

## Reporting a Vulnerability

If you discover a security vulnerability in **Simple-Text-Extractor**, please **do not** open a public issue on GitHub immediately. This could expose users to risk before a fix is available.

Please follow this **responsible disclosure** procedure:
1. Send a detailed email to **app.sencie@gmail.com**.
2. Include "SECURITY" in the subject line.
3. Describe the vulnerability, steps to reproduce it, and the environment used.

### My Commitment

* I will acknowledge receipt of your report within **48 to 72 hours**.
* I will keep you informed of the progress of the fix.
* Once the vulnerability is patched and the update is published on the Microsoft Store, I will publicly credit you (if you wish) in the release notes to thank you for your help.

Thank you for helping keep Simple-Text-Extractor safe for everyone!
